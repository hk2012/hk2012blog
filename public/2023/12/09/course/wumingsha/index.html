# 系统介绍

  • Windows 11 专业版

  • 无名杀官方下载链接

    • https://github.com/libccy/noname/releases/tag/chromium77-client

  • 如果不会翻墙，可以直接下载该文件，但是最好还是去官方下载，以免作者夹带私货

    • Noname-win32-x64.zip

  • 下载之后文件解压后的页面

  • 双击 无名杀.exe 文件后，得到该页面

  • 点击下载无名杀

  • 下载成功后，自动进入页面，该页面是最原始，最纯净的页面，后面的特效需要添加扩展包
